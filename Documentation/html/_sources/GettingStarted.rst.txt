===============
Getting Started
===============

First check whether you have these libraries installed:

* Numpy
* Scipy
* H5py
* Matplotlib


How to Download
---------------

Download directly from `github <https://github.com/nemectad/SFA_f2py/>`_ repository.


How to Run
----------

Run script **SFA_f2py.py** from the original directory by typing the following to the command prompt: ::

    python SFA_f2py.py

Or in the jupyter notebook type: ::

    %run SFA_f2py.py
